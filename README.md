# MegaMartX

MegaMartX is a modern **MERN stack e-commerce application** built with React, Node.js, Express, MongoDB, and Tailwind CSS. It provides a complete online shopping experience with user authentication, product management, and cart functionality.

## Features

- User registration and login
- Browse products with categories
- Add products to cart and manage orders
- Responsive design with Tailwind CSS
- RESTful APIs built with Express and MongoDB

## Tech Stack

- **Frontend:** React, Tailwind CSS, Vite
- **Backend:** Node.js, Express
- **Database:** MongoDB
- **Version Control:** Git & GitHub
